Error

Version

1.0

Descripción

Este método es automáticamente invocado en caso de un error fatal; este simplemte emite el mensaje y para la ejecución. Una clase heredada puede anularlo para personalizar en manejo de errore pero siempre debería parar el script, o elr esultado probablemente sería no válido.

Parámetros

msg
El mensaje de error.
