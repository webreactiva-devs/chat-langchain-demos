SetMargins

SetMargins(float left, float top [, float right])

Version

1.0

Descripción

Define las márgenes izquierda, superior, y derecha. Por defecto, son iguales a 1 cm. Invoque este método para cambiarlas.

Parámetros

left
Márgen izquierda.
top
Márgen superior.
right
Márgen derecha. Valor por defecto es el mismo que el de la izquierda.

Vea además

SetLeftMargin(), SetTopMargin(), SetRightMargin(), SetAutoPageBreak().
