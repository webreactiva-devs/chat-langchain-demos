AddPage

AddPage([string orientation])

Version

1.0

Descripción

Adiciona una nueva página al documento. Si la página ya está presente, el método Footer() es invocado primero para producir el pie de página. Entonces es adicionada la página, la posición actual es puesta en la esquina superior izquierda de acuerdo a las márgenes izquierda y superior, y el Header() es invocado para mostrar el encabezado.
La fuente la cual fue fijada antes de ser invocada es restaurada automáticamente. No es necesario invocar neuvamente SetFont() si desea continuar con la misma fuente. Esto también aplica para colores y ancho de líneas.
El origen del sistema de coordenadas es la esquina superior izquierda y se incrementan hacia abajo.

Parámetros

orientation
Orientación de página. Los vfalores posibles son (case insensitive):
  • P o Portrait
  • L o Landscape
El valor por defecto el mismo que que se ha pasado al constructor.

Vea además

FPDF(), Header(), Footer(), SetMargins().
