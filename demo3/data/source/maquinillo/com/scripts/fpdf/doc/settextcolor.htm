SetTextColor

SetTextColor(int r [, int g, int b])

Version

1.3

Descripción

Define el color usado por el texto. Este puede ser expresado en componentes RGB o escala de grises. El método puede se invocado antes que la primera página sea creada y el valor es retenido de página a página.

Parámetros

r
Si g et b es suministrado, componente rojo; si no, indica el nivel de grises. Valor entre 0 y 255.
g
Compenente Verde (entre 0 y 255).
b
Compenente azul (entre 0 y 255).

Vea además

SetDrawColor(), SetFillColor(), Text(), Cell(), MultiCell().
