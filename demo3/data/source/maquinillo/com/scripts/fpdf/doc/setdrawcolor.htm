SetDrawColor

SetDrawColor(int r [, int g, int b])

Version

1.3

Descripción

Define el color usado para las operaciones de graficación (lineas, rectangulos, y bordes de celdas ). Este puede ser expresado en componentes RGB o en escala de grises. El método puede ser invocado antes que la primera página sea creada y el valor sea retenido de página a página.

Parámetros

r
Si g et b son suministrados, componente rojo; si no, si no indica nivel de grises. Valores entre 0 y 255.
g
Componete Verde (entre 0 y 255)
b
Componete azul (entre 0 y 255)

Vea además

SetFillColor(), SetTextColor(), Line(), Rect(), Cell(), MultiCell().
